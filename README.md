# se-ai-review

## Setup

### 1. Create a Python 3.12.2 environment

```bash
conda create -n se-ai-review python=3.12.2 -y
conda activate se-ai-review
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Shiny app

```bash
shiny run app.py
```
